# Introducción y Objetivos

Este documento describe la arquitectura del sistema ERP para el Grupo X.

## Objetivo

El sistema ERP debe gestionar eficientemente los módulos principales del negocio, facilitando la administración de compras, facturación, stock, activos fijos, empleados y el sistema EIS.

## Requisitos de negocio principales (Módulo de Compras)

- Registrar nuevos productos con información básica.
- Gestionar proveedores y precios unitarios.
- Procesar órdenes de compra.
- Controlar inventarios y movimientos.
- Generar reportes para la gestión.
